## Starlight Issue Wikiの特徴

Starlight Issue Wikiは、GitHubのissueシステムを使ってシンプルながらも強力なWikiサイトを構築するためのテンプレートです。

### 主な特徴

- **GitHubと完全連携**: issueで管理し、変更を自動的に反映
- **カテゴリー分類**: ラベルを使ってコンテンツを自動分類
- **Markdown完全対応**: リッチなコンテンツ作成が可能
- **スマートフォン対応**: あらゆるデバイスで最適な表示

### 使い方

1. [GitHubリポジトリ](https://github.com/nao-amj/starlight-issue-wiki)を開く
2. 「Issues」タブをクリック
3. 「New Issue」ボタンでコンテンツを作成
4. ラベルを付けてカテゴリ分類
